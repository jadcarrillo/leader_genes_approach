get_STRING_species {STRINGdb}R Documentation

get_STRING_species

Description

Returns a data frame with the species (i.e. organisms) that are present in STRING.

Usage

get_STRING_species(version=NULL, species_name=NULL)

Arguments

version

STRING version

species_name

name of the species that you are searching

Value

data frame with the species (i.e. organisms) that are present in STRING.
